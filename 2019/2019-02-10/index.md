# 可悲日記

講一個台男悲歌<br/>
昨天去哥倫比亞人家裡喝酒<br/>
聊自己的經歷和歷史<br/>
跟他們說我在台灣很可悲 來澳洲還是很可悲<br/>
說都沒人找我 我也加不進去別人群體<br/>
哥倫比亞人說 別灰心要加油<br/>
我表示 可是我找一堆人都是被一堆藉口打槍 不然就是下次下次 找日本人不知道幾百次<br/>
剛講完<br/>
